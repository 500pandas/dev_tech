## 作业

现在你学会HTML基础了，那么接下来我们就开始~~编写属于自己的燕山大学官网吧~~

编写一个HTML网页

### 作业要求

- 有自己的网站标题（浏览器标签页上）
- 使用分级的标题标签
- 使用加粗、下划线、删除线和倾斜标签
- 插入至少两张图片，其中一张能正常显示，另一张无法正常显示，用文字替代
- *STFW自学HTML表格的创建方式，并在你的作业中使用*
